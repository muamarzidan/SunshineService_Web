Silabus Pelatihan

  • Pengenalan ke UI/UX Design : Apa itu UI dan UX serta fungsinya (Pengantar UI/UX Design); Pentingnya Desain Pengalaman Pengguna (UX Design).
  • Pemahaman Pengguna (UX) : Observasi dan analisis pengguna; Pembuatan user persona (persona pengguna).
  • Fundamental desain antarmuka pengguna (UI Design) : Pengenalan tools figma dan praktik; Tipografi (Font size, spacing, font weight, dan hierarki tipografi); Warna (Pemilihan warna untuk UI design, primary color, secondary color); Praktik membuat tampilan halaman login.

  • Desain antarmuka pengguna (UI Design) : UI Design; Refrensi Design; Membuat moodboard mobile app 3 halaman.
